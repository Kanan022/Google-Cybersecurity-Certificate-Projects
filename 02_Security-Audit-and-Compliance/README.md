## Project Summary
This project involved conducting an internal security audit and risk assessment for a fictional company (Botium Toys). The goal was to align current security practices with industry standards and regulations to improve the overall security posture.

## Key Skills Demonstrated
* **Risk Assessment:** Identifying vulnerabilities, mapping assets, and classifying risks (High/Medium/Low).
* **Compliance Management:** Assessing adherence to key regulations like **GDPR** (General Data Protection Regulation) and **PCI DSS** (Payment Card Industry Data Security Standard).
* **Security Controls:** Analyzing the existing security environment and recommending new controls across **Administrative, Technical, and Physical** categories.
* **Framework Knowledge:** Application of security principles derived from frameworks like the **NIST Cybersecurity Framework (CSF)**.

## Deliverables in this Folder
* `Botium-Toys-Assessment-Report.docx` - The comprehensive report summarizing the audit findings and remediation recommendations.
* `Controls-and-Compliance-Checklist.docx` - The completed checklist used to assess the company's current security implementation.
* `Scope-and-Goals.docx` - Defines the parameters and objectives of the internal security audit.
